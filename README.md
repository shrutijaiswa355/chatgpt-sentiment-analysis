# chatgpt-sentiment-analysis
This research analyzes Kaggle’s "500k ChatGPT Tweets" dataset to track market sentiment and user adoption trends. By mining 500,000 posts, the study uncovers public attitudes and commercial perceptions regarding ChatGPT's economic impact.

# Sentiment Analysis on ChatGPT Tweets

Analysed 500,000 tweets to understand public opinion on ChatGPT using NLP techniques.

## Tools used
R, tidytext, VADER, ggplot2, dplyr, Topic Modelling 

## What I did
- Collected and cleaned 500,000 tweets
- Applied VADER sentiment analysis to classify positive, negative, neutral opinions
- Used topic modelling to identify key themes in the conversation
- Produced a structured insight report with findings

## Key finding
Majority sentiment was positive, with concerns clustering around privacy and job displacement topics.
